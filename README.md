\f0\fs24 \cf0 # SmarterChild Discord Bot\
\
A Discord bot that recreates the classic AOL SmarterChild experience with modern attitude!\
\
## Features\
- Sarcastic and rude responses\
- Random demonic mode (15% chance)\
- Curses back at you\
- Classic emoticons :-) :-P :-D\
- Uses /chat slash command\
\
## Setup\
\
1. Install dependencies:\
```bash\
   npm install\
```\
\
2. Get your Discord bot token and client ID from Discord Developer Portal\
\
3. For local testing, replace the placeholder values in bot.js\
\
4. For Railway deployment, set environment variables:\
   - `DISCORD_TOKEN`\
   - `CLIENT_ID`\
\
5. Run locally:\
```bash\
   node bot.js\
```\
\
## Usage\
\
In Discord, type: `/chat message: your message here`\
\
## Deploy to Railway\
\
1. Push to GitHub\
2. Connect Railway to your repo\
3. Add environment variables\
4. Deploy!\
\
Enjoy! :-D}
